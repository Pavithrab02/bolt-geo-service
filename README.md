# bolt-geo-service

A prototype developed to understand the working geospatial indexing used in the applications like Uber and Rapido.
# currently only finds the static drivers. Planning on dynamic drivers too.
# Tools used 
- Go for programming
- database: Supabase(postGIS)
- Render for deployment
Have used spatial indexing for **lattitudes and longitude mapping** with **Hexagonal cells** to accurately map the driver coordinates
